# Pinnacle Shield Insurance

## Overview
Pinnacle Shield Insurance is a fictional insurance company website built as a responsive, modern static site. It demonstrates best practices in HTML, CSS (Bootstrap), and JavaScript for a professional business presence.

## Features
- Home page with company highlights and testimonials
- About Us page with company overview, leadership team, and core values
- FAQ page with searchable, filterable accordion (Bootstrap + JS)
- Quote page for insurance inquiries
- Consistent navigation bar and footer on all pages
- Responsive design for all devices
- Modern icons, improved typography, and clean layout
- Automated deployment to GitHub Pages using GitHub Actions

## Tech Stack
- HTML5 & CSS3 (with Bootstrap 5)
- JavaScript (for FAQ search/filter)
- GitHub Actions for CI/CD

## Deployment
This site is automatically deployed to GitHub Pages on every push to the `main` branch using a workflow in `.github/workflows/deploy.yml`.

## How to Run Locally
1. Clone the repository
2. Open `index.html` in your browser

## How to Deploy
1. Commit and push changes to the `main` branch
2. GitHub Actions will build and deploy the site to GitHub Pages
